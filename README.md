# Pure Css Components
[![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg)](https://github.com/felippemauricio/pure-css/blob/master/LICENSE.md)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](https://github.com/felippemauricio/pure-css/pulls)

A set of common UI Components using the power of CSS + HTML and without Javascript.

![](https://raw.githubusercontent.com/felippemauricio/pure-css/master/docs/images/css-html.jpg)

## Components

- [Carousel](https://felippemauricio-pure-css.now.sh/components/carousel)
- [Collapse](https://felippemauricio-pure-css.now.sh/components/collapse)
- [Dropdown](https://felippemauricio-pure-css.now.sh/components/dropdown)
- [Modal](https://felippemauricio-pure-css.now.sh/components/modal)
- [Tab](https://felippemauricio-pure-css.now.sh/components/tab)
- [Tooltip](https://felippemauricio-pure-css.now.sh/components/tooltip)

## Production

Each version of the code that is merged to `master`, the system makes the deploy on `Now`. So, you can access using the `https://felippemauricio-pure-css.now.sh/components` host.
